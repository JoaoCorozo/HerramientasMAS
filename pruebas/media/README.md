## 💡 ¿Cómo grabar la pantalla fácilmente en Windows?

1. **Herramienta de Recortes de Windows (Recomendado):**
   * Presiona `Win + Shift + S` para capturas de pantalla estáticas.
   * O abre la app "Herramienta de Recortes" (Snipping Tool), cámbiala al modo **Grabar Video** (ícono de cámara de video) y graba la acción. Luego puedes usar cualquier convertidor online gratuito para pasarlo a formato `.gif`.

2. **ScreenToGif (Software Gratuito y de Código Abierto):**
   * Es una herramienta excelente y ligera para Windows que te permite grabar un área de la pantalla y exportarla directamente como un archivo `.gif` optimizado.

3. **PowerPoint:**
   * Sí, PowerPoint tiene una función llamada "Grabación de pantalla" en la pestaña *Insertar* que funciona excelente.
